## 🛠 Features

Project Onyx includes several tools to help you organize data.

### Automatic Sorting
The software scans your Downloads folder every hour. It identifies file types like photos, documents, and spreadsheets. It moves these files into folders named Pictures, Documents, and Sheets. You save time by letting the software handle the filing.

### Duplicate Detection
Digital clutter often happens when you save the same file twice. Project Onyx compares file names and sizes. It alerts you when it finds duplicates. You decide whether to delete or keep the extra files.

### Custom Rule Creation
You can set your own rules for how files move. If you want all files from a specific sender to go to a folder named Taxes, the software allows you to define this path. Open the Settings menu to add a custom rule.

## ❓ Troubleshooting

If you run into trouble, try these steps to solve the issue.

### The program fails to open
Restart your computer. Sometimes Windows needs a refresh to recognize a new program. Make sure you have an internet connection during the first launch, as the program might need to confirm the license.

### The software does not sort files
Check the Settings menu. Ensure the "Enable Auto-Sort" box has a checkmark. Verify that you selected folders that the software has permission to access. If you try to sort files in a system folder, Windows might block the action for security reasons. Stick to folders you created yourself for the best results.

### The installation window freezes
Close the window and try again. Right-click the installer and choose "Run as Administrator." This gives the installer permission to create necessary folders on your hard drive.
